# 설계 문서 및 트러블슈팅 리포트 (Docs)

본 디렉터리는 `mini-ipmi` 원격 관리 시스템 프로젝트의 기획, 아키텍처 설계 및 개발 중 직면한 엔지니어링 문제 해결 과정(Troubleshooting)을 상세히 기록한 문서들을 보관합니다.

## 포함된 문서

- **`Embedded_system_remote_control-1.pdf`**
  - 최종 발표 자료로, 원격 제어 시스템의 전체적인 아키텍처 및 구현 결과를 요약한 프레젠테이션 문서입니다.
- **`mini-project-1.pdf`**
  - 개발 초기 단계에 구상한 간단한 시스템 구조와 기본 요구사항이 기술된 기획 문서입니다.
- **`troubleshooting.md`**
  - Bluetooth, TCP Socket, STM32 UART 인터럽트 제어 및 리눅스 커널 통신 규격 차이로 인해 발생했던 버그(데이터 유실, 프리징 등)를 해결한 원인 분석 및 해결 조치 리포트입니다.
